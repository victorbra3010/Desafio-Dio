# Desafio Dio

### Desafio de usar IA pra resolver problemas simples:

1. **Se é Impar ou Par**
2. **Média de notas**
3. **Verificando Palíndromos**




>Objetivo: Desenvolver habilidades no uso da inteligência artificial como um copiloto no processo de programação, familiarizando-se com o ambiente do GitHub Codespaces e consolidando, de forma prática, os principais conhecimentos adquiridos. 

>E claro concluir o desafio da DIO!!
